# Fact-Check Protocol

## Process

1. **Extract** key claims from all Tier 1–2 source extractions
2. **Cross-verify** each claim against ≥2 independent sources
3. **Assign verdict**:

| Verdict      | Criteria                                | Indicator    |
| ------------ | --------------------------------------- | ------------ |
| Confirmed    | 2+ independent sources agree            | CONFIRMED    |
| Partial      | 1 source confirms, others silent        | PARTIAL      |
| Unverified   | No independent verification found       | UNVERIFIED   |
| Contradicted | Independent source directly contradicts | CONTRADICTED |

## Independence Check

Two sources are **independent** when:

- Different authors/organizations
- No shared funding source
- Not citing each other as primary evidence
- Different methodological approach

Two sources are **NOT independent** when:

- One cites the other as sole evidence
- Same research group, different papers
- Both quoting the same primary study

## Verification Priority

1. Claims that inform recommended actions (highest priority)
2. Claims with confidence "Strong" — verify to confirm
3. Claims cited by multiple sources — check for shared origin
4. Quantitative claims — verify numbers specifically

## Red Flags

- Claim repeated widely but traceable to single original source
- Precise statistics without methodology reference
- Universal claims ("always", "never") in complex domains
- Claims contradicting well-established literature without explanation
