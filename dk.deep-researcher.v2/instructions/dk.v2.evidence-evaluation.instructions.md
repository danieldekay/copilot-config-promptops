# Evidence Evaluation

## CRAAP Framework

Score each Tier 1–3 source on 5 dimensions (1–5 scale):

| Dimension     | Key Questions                                                                  |
| ------------- | ------------------------------------------------------------------------------ |
| **Currency**  | When published? Still relevant? (>2yr may be outdated in fast fields)          |
| **Relevance** | Directly addresses question? Right depth? Right audience?                      |
| **Authority** | Author qualified? Institutional backing? Credentials verifiable?               |
| **Accuracy**  | Supported by evidence/data/citations? Peer reviewed? Verifiable?               |
| **Purpose**   | Informational, commercial, persuasive? Biases acknowledged? Funding disclosed? |

## Evidence Confidence Levels

| Level        | Criteria                       | Label                        |
| ------------ | ------------------------------ | ---------------------------- |
| Strong       | ≥3 independent sources, ≥1 T1  | "Evidence strongly supports" |
| Moderate     | 2 sources agree, or 1 T1 alone | "Evidence suggests"          |
| Weak         | Single source, or T3+ only     | "Preliminary evidence"       |
| Contradicted | Sources explicitly disagree    | "Evidence is conflicting"    |
| None         | No sources address this        | "No evidence found"          |

## Contradiction Analysis

When sources disagree:

1. **What**: State the contradiction clearly
2. **Who**: Which sources on each side
3. **Why**: Different methodology? Scope? Time period? Context? Fundamental disagreement?
4. **Resolution**: Can it be resolved? How?
5. **Impact**: Does this affect the research question?

## Bias Detection

Check for: publication bias, funding bias, confirmation bias, survivorship bias, authority bias, recency bias.
